GAMBARAN UMUM
Radar Market adalah database aktivitas bandar per saham yang dihasilkan AFL Scanner dari data broker setiap hari. Berbeda dengan Master Stock yang berisi harga & FA ratios, Radar berisi aliran dana institusi (smart money) — siapa yang beli/jual dan berapa banyak dalam miliar Rp.

Perbedaan Master Stock vs Radar:

Master Stock	Radar Market
Isi	Harga, Volume, FA Ratios	NBS 1D/5D/10D, VWAP, FlowState
Sumber	AFL Scanner harian	AFL Scanner dari data broker
Jumlah saham	~800+ saham	~500-600 saham (yang ada data broker)
Dipakai di	Semua screen	Bandar Detector, Sector Rotation, Screener Smart Money
Dipakai oleh:

Screen	Data Radar yang dipakai
Bandar Detector	NBS 1D/5D/10D · VWAP · FlowState · Signal
StockTools: Smart Money In	nbs1d > 0 AND nbs5d > 0 AND nbs10d > 0
Sector Rotation	avgBandarScore · avgTrendScore · accRatio per sektor
Screener	filter flow_state · nbs_1d_pos/neg
Stock Detail Smart Money tab	bandarScore · trendScore · flowState

FIELD LENGKAP RADAR MARKET
Net Buy/Sell (NBS) — Core Data
Field	Contoh	Keterangan
nbs1d	15.5	Net Buy/Sell 1 hari dalam Miliar Rp
nbs5d	45.2	Net Buy/Sell 5 hari (1 minggu) dalam Miliar Rp
nbs10d	120.8	Net Buy/Sell 10 hari (2 minggu) dalam Miliar Rp
Interpretasi NBS:

Positif (+) = Net Buy → bandar lebih banyak beli dari jual → Accumulation
Negatif (-) = Net Sell → bandar lebih banyak jual dari beli → Distribution
Magnitude = semakin besar absolutnya, semakin besar aktivitas bandar
NBS 1D: +15.5B → bandar net beli 15.5 miliar Rp hari ini
NBS 5D: -45.2B → dalam 5 hari terakhir, net jual 45.2 miliar Rp
NBS 10D: +120.8B → 10 hari terakhir masih net beli kuat

VWAP Bandar
Field	Contoh	Keterangan
vwapD1	10080	Harga rata-rata bandar bertransaksi hari ini
vwap5dAvg	9950	Harga rata-rata bandar bertransaksi 5 hari
Penggunaan VWAP:

Jika harga sekarang > vwapD1 → bandar lagi untung
Jika harga sekarang < vwapD1 → bandar lagi rugi (potensi cut atau hold)
Selisih harga vs vwapD1 = estimasi P/L posisi bandar
Flow State — Sinyal Arah Dana
Field	Nilai yang mungkin	Keterangan
flowState	"Strong Accumulation"	Akumulasi kuat (1D)
"Accumulation"	Akumulasi (1D)
"Neutral"	Sideways / belum jelas
"Distribution"	Distribusi (1D)
"Strong Distribution"	Distribusi kuat (1D)
vwapContext	sama di atas	Context 5 hari
trendContext	sama di atas	Context 10 hari / trend
Mapping ke Signal di Bandar Detector:

function flowToSignal(flowState) {
  const s = (flowState ?? "").toUpperCase();
  if (s.includes("STRONG ACCUMULATION")) return "Accumulation";
  if (s.includes("ACCUMULATION"))        return "Accumulation";
  if (s.includes("STRONG DISTRIBUTION")) return "Distribution";
  if (s.includes("DISTRIBUTION"))        return "Distribution";
  return "Neutral";
}
// signal1d  = flowToSignal(flowState)    → sinyal hari ini
// signal5d  = flowToSignal(vwapContext)  → sinyal 5 hari
// signal10d = flowToSignal(trendContext) → sinyal 10 hari / trend

Score & Momentum
Field	Range	Keterangan
bandarScore	0–100	Kekuatan akumulasi bandar (AFL-computed)
trendScore	0–100	Kekuatan trend teknikal
rsMom	-∞ ~ +∞	Relative Strength Momentum vs IHSG
changePct	%	Perubahan harga hari ini dari radar AFL
rsi	0–100	RSI dari radar AFL
perf10d	%	Performa 10 hari terakhir

function BandarCard({ item, onPress }) {
  const { symbol, name, oneDayBillion, fiveDayBillion, tenDayBillion,
          bandarAvg1d, signal1d, signal5d, signal10d } = item;
  
  // Harga saham saat ini — dari masterData cache
  const stockInfo = stockMap.get(symbol);
  const currentPrice = stockInfo?.close ?? 0;
  
  // P/L posisi bandar vs VWAP hari ini
  const vwapDiff    = currentPrice - bandarAvg1d;
  const vwapDiffPct = bandarAvg1d > 0 ? ((vwapDiff / bandarAvg1d) * 100).toFixed(2) : null;
  
  const signalColor = (sig) => {
    if (sig === "Accumulation")  return "#34d399";  // hijau
    if (sig === "Distribution")  return "#f87171";  // merah
    return "#94a3b8";                               // abu
  };
  
  const signalIcon = (sig) => {
    if (sig === "Accumulation")  return "🟢";
    if (sig === "Distribution")  return "🔴";
    return "⚪";
  };
  
  const formatBn = (val) => {
    const v = val ?? 0;
    const sign = v >= 0 ? "+" : "";
    return `${sign}${v.toFixed(1)}B`;
  };
  
  return (
    <TouchableOpacity onPress={() => onPress(symbol)}
      style={{ backgroundColor: "#1e2433", borderRadius: 12, padding: 14, marginBottom: 10 }}>
      
      {/* Header: Ticker + Signal Badge */}
      <View style={{ flexDirection: "row", justifyContent: "space-between", alignItems: "center" }}>
        <View>
          <Text style={{ color: "white", fontWeight: "900", fontSize: 16 }}>{symbol}</Text>
          <Text style={{ color: "#94a3b8", fontSize: 11 }} numberOfLines={1}>{name}</Text>
        </View>
        <View style={{ backgroundColor: signalColor(signal1d) + "20",
          borderRadius: 8, paddingHorizontal: 10, paddingVertical: 4 }}>
          <Text style={{ color: signalColor(signal1d), fontWeight: "700", fontSize: 12 }}>
            {signalIcon(signal1d)} {signal1d}
          </Text>
        </View>
      </View>
      
      {/* Harga & VWAP bandar */}
      <View style={{ flexDirection: "row", justifyContent: "space-between", marginTop: 10 }}>
        <Text style={{ color: "#60a5fa", fontSize: 12 }}>
          Harga: {currentPrice.toLocaleString("id-ID")}
        </Text>
        {vwapDiffPct && (
          <Text style={{ color: parseFloat(vwapDiffPct) >= 0 ? "#34d399" : "#f87171", fontSize: 11 }}>
            Bandar VWAP: {bandarAvg1d.toLocaleString("id-ID")}
            {" "}({vwapDiffPct}%)
          </Text>
        )}
      </View>
      
      {/* NBS Multi-Timeframe Grid */}
      <View style={{ flexDirection: "row", marginTop: 10, gap: 8 }}>
        {[
          { label: "1D", value: oneDayBillion, signal: signal1d },
          { label: "5D", value: fiveDayBillion, signal: signal5d },
          { label: "10D", value: tenDayBillion, signal: signal10d },
        ].map((tf) => (
          <View key={tf.label} style={{ flex: 1, backgroundColor: "#0f1629",
            borderRadius: 8, padding: 8, alignItems: "center" }}>
            <Text style={{ color: "#64748b", fontSize: 10 }}>{tf.label}</Text>
            <Text style={{
              color: (tf.value ?? 0) >= 0 ? "#34d399" : "#f87171",
              fontWeight: "700", fontSize: 13, marginTop: 2
            }}>
              {formatBn(tf.value)}
            </Text>
            <Text style={{ color: signalColor(tf.signal), fontSize: 9, marginTop: 2 }}>
              {signalIcon(tf.signal)} {tf.signal === "Accumulation" ? "ACC" :
                tf.signal === "Distribution" ? "DIST" : "NEU"}
            </Text>
          </View>
        ))}
      </View>
    </TouchableOpacity>
  );
}

BAGIAN 2: KONSISTENSI MULTI-TIMEFRAME
Ini cara menilai kekuatan sinyal bandar berdasarkan konsistensi 3 timeframe:

function getBandarStrength(signal1d, signal5d, signal10d) {
  const signals = [signal1d, signal5d, signal10d];
  const accCount  = signals.filter(s => s === "Accumulation").length;
  const distCount = signals.filter(s => s === "Distribution").length;
  
  if (accCount === 3) return { label: "ACC KUAT", color: "#10b981", icon: "🟩🟩🟩" };
  if (accCount === 2) return { label: "ACC",      color: "#34d399", icon: "🟩🟩⬜" };
  if (distCount === 3) return { label: "DIST KUAT", color: "#dc2626", icon: "🟥🟥🟥" };
  if (distCount === 2) return { label: "DIST",     color: "#f87171", icon: "🟥🟥⬜" };
  return { label: "MIXED",   color: "#94a3b8", icon: "⬜⬜⬜" };
}

function SmartMoneyTab({ symbol, currentPrice }) {
  const radarInfo = radarMap.get(symbol);
  if (!radarInfo) return <NoDataPlaceholder label="Data Smart Money belum tersedia" />;
  
  const flowSignal = flowToSignal(radarInfo.flowState ?? "");
  const flowColor = flowSignal === "Accumulation" ? "#34d399"
                  : flowSignal === "Distribution" ? "#f87171" : "#94a3b8";
  
  const vwapDiff = currentPrice && radarInfo.vwapD1
    ? currentPrice - radarInfo.vwapD1 : null;
  
  return (
    <ScrollView>
      {/* Score Bars */}
      <View style={{ backgroundColor: "#1e2433", borderRadius: 12, padding: 14, marginBottom: 10 }}>
        <Text style={{ color: "#94a3b8", fontSize: 12, marginBottom: 8 }}>Kekuatan Bandar</Text>
        <ScoreBar label="Bandar Score" value={radarInfo.bandarScore ?? 0} color="#60a5fa" />
        <ScoreBar label="Trend Score"  value={radarInfo.trendScore  ?? 0} color="#a78bfa" />
      </View>
      
      {/* Flow State */}
      <View style={{ backgroundColor: "#1e2433", borderRadius: 12, padding: 14, marginBottom: 10 }}>
        <Text style={{ color: "#94a3b8", fontSize: 12 }}>Flow State</Text>
        <View style={{ flexDirection: "row", alignItems: "center", marginTop: 6 }}>
          <View style={{ width: 10, height: 10, borderRadius: 5, backgroundColor: flowColor, marginRight: 8 }} />
          <Text style={{ color: flowColor, fontWeight: "700", fontSize: 14 }}>
            {radarInfo.flowState ?? "N/A"}
          </Text>
        </View>
      </View>
      
      {/* NBS Grid */}
      <View style={{ backgroundColor: "#1e2433", borderRadius: 12, padding: 14, marginBottom: 10 }}>
        <Text style={{ color: "#94a3b8", fontSize: 12, marginBottom: 8 }}>Net Buy/Sell (Miliar Rp)</Text>
        <View style={{ flexDirection: "row", gap: 8 }}>
          {[
            { label: "1D",  value: radarInfo.nbs1d },
            { label: "5D",  value: radarInfo.nbs5d },
            { label: "10D", value: radarInfo.nbs10d },
          ].map(tf => (
            <View key={tf.label} style={{ flex: 1, backgroundColor: "#0f1629",
              borderRadius: 8, padding: 10, alignItems: "center" }}>
              <Text style={{ color: "#64748b", fontSize: 10 }}>{tf.label}</Text>
              <Text style={{
                color: (tf.value ?? 0) >= 0 ? "#34d399" : "#f87171",
                fontWeight: "700", fontSize: 14, marginTop: 4
              }}>
                {tf.value != null ? `${tf.value >= 0 ? "+" : ""}${tf.value.toFixed(1)}B` : "N/A"}
              </Text>
            </View>
          ))}
        </View>
      </View>
      
      {/* VWAP Analysis */}
      {radarInfo.vwapD1 != null && (
        <View style={{ backgroundColor: "#1e2433", borderRadius: 12, padding: 14, marginBottom: 10 }}>
          <Text style={{ color: "#94a3b8", fontSize: 12, marginBottom: 8 }}>VWAP Bandar</Text>
          <View style={{ flexDirection: "row", justifyContent: "space-between" }}>
            <View>
              <Text style={{ color: "#64748b", fontSize: 10 }}>VWAP 1D</Text>
              <Text style={{ color: "white", fontWeight: "700" }}>
                {radarInfo.vwapD1.toLocaleString("id-ID")}
              </Text>
            </View>
            {radarInfo.vwap5dAvg != null && (
              <View>
                <Text style={{ color: "#64748b", fontSize: 10 }}>VWAP 5D</Text>
                <Text style={{ color: "white", fontWeight: "700" }}>
                  {radarInfo.vwap5dAvg.toLocaleString("id-ID")}
                </Text>
              </View>
            )}

            <Text style={{ color: vwapDiff >= 0 ? "#34d399" : "#f87171", fontSize: 12, marginTop: 8 }}>
              Harga {vwapDiff >= 0 ? "di atas" : "di bawah"} VWAP bandar: {Math.abs(vwapDiff).toLocaleString("id-ID")} poin
              {" → Bandar sedang {vwapDiff >= 0 ? "untung" : "rugi"}"}

  );
}

// Konversi flowState ke display yang ramah user
function getFlowLabel(flowState) {
  const s = (flowState ?? "").toUpperCase();
  if (s.includes("STRONG ACCUMULATION")) return { label: "Akumulasi Kuat", color: "#10b981", short: "ACC ✦" };
  if (s.includes("ACCUMULATION"))        return { label: "Akumulasi",      color: "#34d399", short: "ACC" };
  if (s.includes("STRONG DISTRIBUTION")) return { label: "Distribusi Kuat", color: "#dc2626", short: "DIST ✦" };
  if (s.includes("DISTRIBUTION"))        return { label: "Distribusi",     color: "#f87171", short: "DIST" };
  return { label: "Netral", color: "#94a3b8", short: "NEU" };
}
// NBS bar width untuk visualisasi
function nbsBarWidth(value, maxValue) {
  if (maxValue === 0) return 0;
  return Math.min(100, (Math.abs(value) / Math.max(maxValue, 1)) * 100);
}
// Interpretasi VWAP vs harga sekarang
function vwapInterpretation(currentPrice, vwap) {
  if (!vwap || !currentPrice) return null;
  const diffPct = ((currentPrice - vwap) / vwap) * 100;
  if (diffPct > 2)   return { text: "Harga jauh di atas VWAP bandar (bandar untung besar)", color: "#34d399" };
  if (diffPct > 0)   return { text: "Harga di atas VWAP bandar (bandar untung)", color: "#6ee7b7" };
  if (diffPct > -2)  return { text: "Harga di bawah VWAP bandar (bandar sedikit rugi)", color: "#fca5a5" };
  return { text: "Harga jauh di bawah VWAP bandar (bandar rugi — potensi cut loss)", color: "#f87171" };
}
// Format NBS dengan warna
function formatNBS(value) {
  if (value == null) return { text: "N/A", color: "#64748b" };
  const sign  = value >= 0 ? "+" : "";
  const color = value >= 0 ? "#34d399" : "#f87171";
  const text  = `${sign}${value.toFixed(1)}B`;
  return { text, color };
}
