# Program Afiliasi — Panduan Implementasi Mobile
## Dua tipe afiliasi, threshold subscriber, dan progress bar

---

## MODEL BISNIS AFILIASI

| | Tipe 1 — Community Leader | Tipe 2 — One Time |
|--|--------------------------|-------------------|
| `commissionType` | `"percentage"` | `"one_time"` |
| Komisi | **30% recurring** per bulan | **40% sekali** saja |
| Syarat aktif | **Min 30 subscriber aktif** | Tidak ada |
| Jika < 30 aktif | Komisi dibekukan, admin review | N/A |
| Bukti apply | **Wajib** — link/desc group | Tidak wajib |
| Target afiliasi | Pengelola komunitas 50+ member | Siapa saja |
| Upgrade | Bisa upgrade ke Type 1 | — |

Diskon untuk referred user: **20%** di kedua tipe.

## FLOW SCREEN

```
Buka Afiliasi
     │
     ├── hasProfile: false → Screen Pilih Tipe + Form Apply
     │
     └── hasProfile: true
            ├── status: "pending"  → Dashboard + Banner Menunggu
            └── status: "active"   → Dashboard Lengkap
                    ├── [Type 1] Progress bar subscriber + badge frozen/aktif
                    ├── Kode referral + share
                    ├── Statistik (komisi, referral, saldo)
                    ├── Riwayat komisi
                    └── Request Payout (blocked jika frozen)
```

## ENDPOINT SUMMARY

| Method | URL | Auth | Keterangan |
|--------|-----|------|-----------|
| `GET`  | `/api/affiliate/me` | ✅ | Profil + stats + quality |
| `POST` | `/api/affiliate/apply` | ✅ | Daftar — pilih tipe + bukti group |
| `POST` | `/api/affiliate/payout-request` | ✅ | Pencairan (blocked jika frozen) |
| `GET`  | `/api/affiliate/commission-types` | ❌ | Publik |

## SCREEN: Form Apply

```javascript
function FormApplyScreen({ tipe, onSuccess }) {
  const isType1 = tipe === "percentage";
  const [marketReach,   setMarketReach]   = useState("");
  const [groupProof,    setGroupProof]    = useState("");
  const [preferredCode, setPreferredCode] = useState("");
  const [loading,       setLoading]       = useState(false);
  const [error,         setError]         = useState("");

  async function handleApply() {
    if (!marketReach.trim()) { setError("Deskripsi pangsa pasar wajib."); return; }
    if (isType1 && !groupProof.trim()) { setError("Link/bukti group wajib untuk Community Leader."); return; }

    setLoading(true); setError("");
    try {
      const res = await fetchWithAuth("/api/affiliate/apply", {
        method:  "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          marketReach:    marketReach.trim(),
          commissionType: tipe,
          groupProof:     isType1 ? groupProof.trim() : undefined,
          preferredCode:  preferredCode.trim() || undefined,
        }),
      });
      const json = await res.json();
      if (!res.ok) { setError(json.error ?? "Gagal mendaftar."); return; }
      onSuccess();
    } catch { setError("Terjadi kesalahan."); }
    finally { setLoading(false); }
  }

  return (
    <ScrollView style={{ flex: 1, backgroundColor: "#060e1f", padding: 20 }}>
      {/* Badge tipe */}
      <View style={[styles.tipeBanner,
        { backgroundColor: isType1 ? "rgba(245,158,11,0.1)" : "rgba(59,130,246,0.1)" }]}>
        <Text style={{ color: isType1 ? "#f59e0b" : "#3b82f6", fontWeight: "700" }}>
          {isType1 ? "👑 Community Leader — 30% Recurring" : "⚡ One Time — 40%"}
        </Text>
      </View>

      <Text style={styles.label}>Deskripsi Pangsa Pasar *</Text>
      <TextInput style={styles.textArea} multiline numberOfLines={4}
        placeholder="Contoh: Saya mengelola grup Telegram saham IDX dengan 500+ member aktif..."
        placeholderTextColor="#475569" value={marketReach} onChangeText={setMarketReach} />

      {isType1 && (
        <>
          <Text style={styles.label}>Link / Bukti Komunitas *</Text>
          <TextInput style={styles.input}
            placeholder="https://t.me/nama_group atau link YouTube/WA..."
            placeholderTextColor="#475569" value={groupProof} onChangeText={setGroupProof} />
          <Text style={styles.hint}>Link grup Telegram, channel YouTube, atau screenshot komunitas.</Text>
        </>
      )}

      <Text style={styles.label}>Kode Afiliasi Pilihan (opsional)</Text>
      <TextInput style={styles.input} autoCapitalize="characters"
        placeholder="Contoh: BUDI2024" placeholderTextColor="#475569" maxLength={20}
        value={preferredCode}
        onChangeText={t => setPreferredCode(t.toUpperCase().replace(/[^A-Z0-9]/g, ""))} />
      <Text style={styles.hint}>Huruf kapital & angka. Kosong = otomatis dari username.</Text>

      {error ? <Text style={styles.error}>⚠ {error}</Text> : null}

      <Pressable style={[styles.btnPrimary, loading && styles.btnDisabled]}
        onPress={handleApply} disabled={loading}>
        <Text style={styles.btnPrimaryText}>
          {loading ? "Mengirim..." : "Kirim Aplikasi"}
        </Text>
      </Pressable>
    </ScrollView>
  );
}
```

## DASHBOARD: Komponen Progress Bar Threshold (Type 1)

Tampilkan di paling atas dashboard untuk afiliasi Type 1.

```javascript
function ThresholdBar({ activeUsers, threshold = 30, frozen }) {
  const pct = Math.min((activeUsers / threshold) * 100, 100);
  const met = activeUsers >= threshold;

  return (
    <View style={styles.thresholdCard}>
      <View style={{ flexDirection: "row", justifyContent: "space-between", marginBottom: 8 }}>
        <Text style={styles.thresholdTitle}>Subscriber Aktif</Text>
        <View style={[styles.statusBadge,
          { backgroundColor: met
            ? "rgba(34,197,94,0.15)" : "rgba(239,68,68,0.15)" }]}>
          <Text style={{ color: met ? "#22c55e" : "#ef4444",
            fontSize: 11, fontWeight: "700" }}>
            {met ? "✓ KOMISI AKTIF" : "✗ KOMISI BEKU"}
          </Text>
        </View>
      </View>

      {/* Progress bar */}
      <View style={styles.progressBg}>
        <View style={[styles.progressFill, {
          width: `${pct}%`,
          backgroundColor: met ? "#22c55e" : "#f59e0b",
        }]} />
      </View>

      <View style={{ flexDirection: "row", justifyContent: "space-between", marginTop: 6 }}>
        <Text style={{ color: "#94a3b8", fontSize: 12 }}>
          <Text style={{ color: met ? "#22c55e" : "#f59e0b",
            fontWeight: "800", fontSize: 16 }}>{activeUsers}</Text>
          {" "}dari{" "}
          <Text style={{ fontWeight: "700" }}>{threshold}</Text>
          {" "}subscriber aktif
        </Text>
        <Text style={{ color: "#64748b", fontSize: 11 }}>
          {threshold - activeUsers > 0 ? `Butuh ${threshold - activeUsers} lagi` : "Threshold tercapai"}
        </Text>
      </View>

      {frozen && (
        <View style={styles.frozenBanner}>
          <Text style={{ color: "#fbbf24", fontSize: 12 }}>
            ⚠ Komisi dibekukan sementara. Tambah subscriber atau hubungi admin untuk review.
          </Text>
        </View>
      )}
    </View>
  );
}
```

```javascript
function DashboardAfiliasiScreen({ data }) {
  const { profile, quality, referrals, payouts, totalReferrals } = data;
  const isType1 = profile.commissionType === "percentage";

  return (
    <ScrollView style={{ flex: 1, backgroundColor: "#060e1f" }}>
      <View style={styles.header}>
        <Text style={styles.headerTitle}>Program Afiliasi</Text>
        <Text style={{ color: "#64748b", fontSize: 12 }}>
          {isType1 ? "👑 Community Leader" : "⚡ One Time"} · {profile.commissionRate}%
        </Text>
      </View>

      {/* Status pending */}
      {profile.status === "pending" && <PendingBanner />}

      {/* Progress bar threshold — hanya Type 1 */}
      {isType1 && profile.status === "active" && (
        <ThresholdBar
          activeUsers={quality.activeUsers}
          threshold={quality.thresholdRequired}
          frozen={quality.commissionFrozen}
        />
      )}

      {/* Kode referral */}
      <ReferralCodeCard code={profile.code} discountPct={profile.discountPct} />

      {/* Stats */}
      <StatsRow quality={quality} profile={profile} totalReferrals={totalReferrals} />

      {/* Komisi bulan ini */}
      <EarningsCard quality={quality} frozen={quality.commissionFrozen} />

      {/* Riwayat komisi */}
      <ReferralHistory referrals={referrals} />

      {/* Payout */}
      {profile.status === "active" && (
        <PayoutSection profile={profile} payouts={payouts}
          frozen={quality.commissionFrozen && isType1} />
      )}
    </ScrollView>
  );
}
```

```javascript
function PayoutSection({ profile, payouts, frozen }) {
  const [showForm, setShowForm] = useState(false);
  const pending = parseFloat(profile.pendingBalance ?? "0");

  return (
    <View style={{ marginHorizontal: 16, marginBottom: 24 }}>
      <Text style={styles.sectionTitle}>PENCAIRAN KOMISI</Text>

      {/* Saldo pending */}
      <View style={styles.balanceCard}>
        <Text style={{ color: "#64748b", fontSize: 11 }}>SALDO TERSEDIA</Text>
        <Text style={{ color: "#22c55e", fontWeight: "900", fontSize: 26, marginTop: 4 }}>
          Rp {pending.toLocaleString("id-ID")}
        </Text>
        <Text style={{ color: "#475569", fontSize: 11, marginTop: 4 }}>
          Min. penarikan Rp 50.000 · Proses 1-3 hari kerja
        </Text>
      </View>

      {/* Frozen warning untuk Type 1 */}
      {frozen && (
        <View style={styles.frozenBanner}>
          <Text style={{ color: "#fbbf24", fontSize: 12 }}>
            ⚠ Pencairan tidak tersedia saat komisi dibekukan. Hubungi admin.
          </Text>
        </View>
      )}

      {/* Tombol request */}
      {!frozen && pending >= 50000 && (
        <Pressable style={styles.btnPrimary} onPress={() => setShowForm(true)}>
          <Text style={styles.btnPrimaryText}>Request Pencairan</Text>
        </Pressable>
      )}

      {/* Riwayat payout */}
      {payouts?.length > 0 && (
        <View style={{ marginTop: 16 }}>
          <Text style={styles.sectionTitle}>RIWAYAT PENCAIRAN</Text>
          {payouts.map((p) => (
            <View key={p.id} style={styles.payoutRow}>
              <View>
                <Text style={{ color: "#e2e8f0", fontWeight: "600" }}>
                  Rp {parseFloat(p.amount).toLocaleString("id-ID")}
                </Text>
                <Text style={{ color: "#64748b", fontSize: 11 }}>
                  {p.bankName} · {new Date(p.createdAt).toLocaleDateString("id-ID")}
                </Text>
              </View>
              <View style={[styles.payoutBadge,
                { backgroundColor: p.status === "paid"
                  ? "rgba(34,197,94,0.15)" : "rgba(245,158,11,0.15)" }]}>
                <Text style={{ color: p.status === "paid" ? "#22c55e" : "#f59e0b",
                  fontSize: 11, fontWeight: "700" }}>
                  {p.status === "paid" ? "Cair" : "Proses"}
                </Text>
              </View>
            </View>
          ))}
        </View>
      )}

      {/* Modal Form Payout */}
      <ModalPayout visible={showForm} profile={profile}
        onClose={() => setShowForm(false)} />
    </View>
  );
}

function ModalPayout({ visible, profile, onClose }) {
  const [amount,  setAmount]  = useState("");
  const [bank,    setBank]    = useState("");
  const [accNum,  setAccNum]  = useState("");
  const [accName, setAccName] = useState("");
  const [loading, setLoading] = useState(false);

  async function submit() {
    const amt = Number(amount);
    if (!amt || !bank || !accNum || !accName) {
      Alert.alert("Lengkapi semua field"); return;
    }
    if (amt < 50000) { Alert.alert("Minimum Rp 50.000"); return; }

    setLoading(true);
    try {
      const res = await fetchWithAuth("/api/affiliate/payout-request", {
        method:  "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          amount: amt, bankName: bank.trim(),
          accountNumber: accNum.trim(), accountName: accName.trim(),
        }),
      });
      const json = await res.json();
      if (!res.ok) { Alert.alert("Gagal", json.error); return; }
      Alert.alert("Berhasil", json.message);
      onClose();
    } catch { Alert.alert("Terjadi kesalahan"); }
    finally { setLoading(false); }
  }

  return (
    <Modal visible={visible} animationType="slide" transparent>
      <View style={styles.overlay}>
        <View style={styles.sheet}>
          <Text style={styles.sheetTitle}>Request Pencairan</Text>
          <TextInput style={styles.input} placeholder="Jumlah (Rp)"
            placeholderTextColor="#475569" keyboardType="numeric"
            value={amount} onChangeText={setAmount} />
          <TextInput style={styles.input} placeholder="Nama Bank (BCA, BRI, dll)"
            placeholderTextColor="#475569" value={bank} onChangeText={setBank} />
          <TextInput style={styles.input} placeholder="Nomor Rekening"
            placeholderTextColor="#475569" keyboardType="numeric"
            value={accNum} onChangeText={setAccNum} />
          <TextInput style={styles.input} placeholder="Nama Pemilik Rekening"
            placeholderTextColor="#475569" value={accName} onChangeText={setAccName} />
          <View style={styles.row2}>
            <Pressable style={[styles.btn, styles.btnCancel]} onPress={onClose}>
              <Text style={styles.btnCancelText}>Batal</Text>
            </Pressable>
            <Pressable style={[styles.btn, styles.btnSave]} onPress={submit}
              disabled={loading}>
              <Text style={styles.btnSaveText}>
                {loading ? "Mengirim..." : "Konfirmasi"}
              </Text>
            </Pressable>
          </View>
        </View>
      </View>
    </Modal>
  );
}
```

## RINGKASAN PERUBAHAN DARI VERSI SEBELUMNYA

| Hal | Sebelum | Sekarang |
|-----|---------|----------|
| Tipe komisi | 1 tipe (20%) | 2 tipe: Type 1 (30%) / Type 2 (40%) |
| Syarat apply Type 1 | Tidak ada | Wajib bukti komunitas |
| Threshold | Retention 70% | Min 30 subscriber aktif |
| Freeze komisi | Tidak ada | Otomatis jika < 30 aktif |
| Form apply | 1 langkah | Pilih tipe dulu → form |
| Payout jika frozen | Bebas | Diblokir sampai review admin |
