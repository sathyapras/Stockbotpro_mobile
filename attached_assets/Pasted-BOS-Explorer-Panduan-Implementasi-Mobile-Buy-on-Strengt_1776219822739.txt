Buy on Strength Signal Screen
Layar posisi terbuka & sinyal momentum naik dari AFL BuyOnStrength Explorer

GAMBARAN UMUM
Buy on Strength (BOS) adalah layar yang menampilkan portofolio sinyal aktif dari AFL BuyOnStrength Explorer — saham yang sedang dalam posisi terbuka (HOLD), baru mendapat sinyal beli (BUY), atau baru keluar posisi (SELL). Data diupload harian oleh admin via CSV.

Bedanya dengan Stockpick biasa:

Stockpick	BOS Explorer
Data	Real-time API + screener	CSV upload harian oleh admin
Konten	Kandidat entry	Posisi nyata yang sudah/sedang diambil
Tab	Single list	3 tab: BUY / HOLD / SELL
Field	Harga, signal, strength	Entry, SL, TP, G/L, Hold days, SL_Phase
Sumber data: Upload CSV dari AmiBroker AFL Explorer oleh admin → disimpan di server → diambil via API.

Identifikasi
Field	Tipe	Contoh	Keterangan

trend	string	"Weak Bullish +"	Trend dari AFL
close	number	2450	Harga penutupan terakhir (Rp)
Entry & Risk Management
Field	Tipe	Contoh	Keterangan
entry	number	2460	Harga beli yang direkomendasikan
stopLoss	number	2340	Level stop loss
slPct	number	4.9	Jarak SL dari entry dalam %
tp1	number	2580	Target profit 1
tp2	number	2660	Target profit 2
sellPrice	number	0	Harga jual aktual (0 = masih open)
rr	number	1.5	Risk:Reward ratio (dihitung otomatis)
Catatan slPct negatif: Jika slPct < 0, artinya StopLoss sudah di atas Entry — trailing stop sudah bergerak naik mengunci profit. Ini bukan error, ini kondisi positif.

Contoh: slPct = -5.4, entry = 840, stopLoss = 885 → SL sudah di atas entry, posisi profit terlindungi.

KLASIFIKASI SIGNAL
Engine AFL menghasilkan beberapa varian teks signal. Mobile harus memetakannya ke 3 tab:

→ Tab BUY
Signal yang mengandung "BUY" (case-insensitive):

const isBuy = signal.toUpperCase().includes("BUY");
// Contoh: "** BUY **", "Signal BUY", "BUY ON STRENGTH"

→ Tab HOLD
Signal yang mengandung "HOLD":

const isHold = signal.toUpperCase().includes("HOLD");
// Contoh: "HOLD", "Signal HOLD"

→ Tab SELL
Signal yang mengandung "SELL" atau "SOLD":

const isSell = signal.toUpperCase().includes("SELL") ||
               signal.toUpperCase().includes("SOLD");
// Contoh: "SELL", "Signal SELL", "SOLD"

Badge SL_Phase
Kondisi	Warna	Label
Mengandung "P1"	Abu-abu	P1: Fixed SL
Mengandung "TGT1 HIT"	Biru	🎯 TP1 Hit · Trail ON
Mengandung "Trail"	Cyan	🔒 Trail Active
Mengandung "TGT2 HIT"	Ungu	🏆 TP2 Hit · Locked
function getSLPhaseBadge(slPhase: string) {
  if (!slPhase) return null;
  const p = slPhase.toUpperCase();
  if (p.includes("TGT2 HIT")) return { label: "🏆 TP2 Hit · Locked", color: "purple" };
  if (p.includes("TGT1 HIT")) return { label: "🎯 TP1 Hit · Trail ON", color: "blue" };
  if (p.includes("TRAIL"))    return { label: "🔒 Trail Active", color: "cyan" };
  if (p.includes("P1"))       return { label: "P1: Fixed SL", color: "gray" };
  return { label: slPhase, color: "gray" };
}

function getVwapStatus(vwapFilter: string) {
  const v = vwapFilter.toUpperCase();
  if (v.includes("BLOKIR")) return { icon: "🚫", color: "red",    label: "VWAP BLOKIR" };
  if (v.includes("FILTER")) return { icon: "⚠️", color: "orange", label: "VWAP FILTER" };
  if (v.includes("OK"))     return { icon: "✅", color: "green",  label: "VWAP OK" };
  return { icon: "—", color: "gray", label: vwapFilter };
}

data.filter(isBuy).sort((a, b) => (b.glPct ?? 0) - (a.glPct ?? 0))

data.filter(isHold).sort((a, b) => (b.glPct ?? 0) - (a.glPct ?? 0))

data.filter(isSell).sort((a, b) => (b.glPct ?? 0) - (a.glPct ?? 0))

LOGIKA TAMPIL "SL SUDAH NAIK"
Jika slPct < 0, artinya trailing stop sudah bergerak di atas entry — tampilkan indikator positif:

const isTrailingAboveEntry = row.slPct < 0;
// Di card:
if (isTrailingAboveEntry) {
  // Tampilkan "SL naik ↑ Rp X.XXX" dengan teks hijau
  // Bukan merah — ini adalah kondisi profit protection
}

Contoh tampilan:

SL: Rp 885  ↑ di atas entry (profit protected)   ← hijau

vs kondisi normal:

SL: Rp 2.340  (4.9% dari entry)                  ← abu-abu/merah

LOGIKA "TARGET SUDAH TERCAPAI"
function getTargetStatus(row: BosRow) {
  const phase = (row.slPhase ?? "").toUpperCase();
  const tp2Hit = phase.includes("TGT2 HIT");
  const tp1Hit = phase.includes("TGT1 HIT") || tp2Hit;
  return {
    tp1Hit,
    tp2Hit,
    trailActive: phase.includes("TRAIL"),
    lockedAt: tp2Hit
      ? row.slPhase.match(/TS=(\d+)/)?.[1] ?? null
      : null,
  };
}

Gunakan ini untuk menampilkan badge ✅ di samping TP1/TP2.
